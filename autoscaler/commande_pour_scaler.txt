Vous devez modifier le fichier python pour ajouter un bout de code intensif qui tourne en boucle dans une boucle while par exemple.
Ensuite vous devez créer un autoscaler horizontal (HPA) avec un nouveau fichier yaml qui va automatiquement créer de nouveaux pods lorsque la consommation
CPU sera trop élevée.

Votre HorizontalPodAutoscaler devra créer au moins 1 pod, et maximum 10 pods.

Le averageUtilization doit etre 50.

Vous devez aussi créer un deployment pour votre application python, et un service de type NodePort qui va couvrir votre deployment.
Une fois que tout est en place, curl vers votre application plusieurs fois jusqu'a ce que sa consommation CPU soit elevee.
Si tout est bien configuré vous devriez voir de nouveaux pods se créer automatiquement grace au HPA.
